Give your Fastmail account a serious MCP interface.

Fastmail MCP Server connects MCP-compatible assistants to email, calendars, contacts, and mailbox operations through a focused TypeScript server with stdio and HTTP transport support.

npm install @jordonh19/fastmail-mcp-server

FASTMAIL_API_TOKEN="your-api-token" \

# HTTP mode
FASTMAIL_API_TOKEN="your-api-token" \
npx @jordonh19/fastmail-mcp-server --transport http --port 3000

Quick Start

Minimal setup, clean transport model.

1. Create a Fastmail API token

Generate a token with Mail, Calendars, and Contacts access from Fastmail settings.

2. Run the server

Use stdio for local MCP clients, or switch to HTTP with --transport http and --port.

3. Drop it into your client config

{
  "servers": {
    "fastmail": {
      "command": "npx",
      "args": ["@jordonh19/fastmail-mcp-server"],
      "env": {
        "FASTMAIL_API_TOKEN": "your-api-token-here"
      }
    }
  }
}

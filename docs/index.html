Fastmail · MCP · JMAP

Fastmail email, calendar, and contacts for MCP clients.

Fastmail MCP Server is an open-source Model Context Protocol server for Claude Desktop, VS Code, Cursor, Windsurf, and custom MCP clients that need reliable Fastmail access over JMAP.

Use one package to search inboxes, send mail, manage drafts, browse contacts, coordinate calendars, and automate mailbox workflows without building a custom Fastmail integration from scratch. Version 1.1.0 adds label tools, mailbox and account summaries, attachment listing, and setup diagnostics.

Quick Start

Minimal setup, clean transport model.

Fastmail MCP Server is intentionally simple to install: create a Fastmail API token, run the npm package, and point your MCP client at it. The same package works for personal inbox workflows, team assistants, and custom JMAP-based automations.

  1. 1

    Create a Fastmail API token

    Generate a token with Mail, Calendars, and Contacts access from your Fastmail account settings.

  2. 2

    Run the server from npm

    Use stdio for local MCP clients, or switch to HTTP with --transport http and --port.

  3. 3

    Add it to your MCP client config

    {
  "servers": {
    "fastmail": {
      "command": "npx",
      "args": ["@jordonh19/fastmail-mcp-server"],
      "env": {
        "FASTMAIL_API_TOKEN": "your-api-token-here"
      }
    }
  }
}

FAQ

Answers to the queries people search for first.

What is Fastmail MCP Server?

It is an open-source MCP server for Fastmail that exposes email, calendar, contacts, mailbox, identity, diagnostics, analytics, and sampling-oriented tools over the Fastmail JMAP API.

Which MCP clients work with it?

Claude Desktop, VS Code, Cursor, Windsurf, and other MCP-compatible clients can run it over stdio, and custom agents can use the HTTP transport.

Do I need a Fastmail API token?

Yes. You need a Fastmail API token with Mail, Calendars, and Contacts access so the server can authenticate against the JMAP session endpoint.

Can it both read and send email?

Yes. It supports searching and reading mail, replying, forwarding, creating drafts, sending drafts, moving mail, adding and removing labels, updating flags, archiving, deleting, downloading attachments, and inspecting mailbox or account summaries.

Does it support calendars and contacts too?

Yes. The server includes tools for listing calendars, querying events, creating and updating events, and reading or modifying contacts in address books.

Is there an HTTP mode?

Yes. stdio is the default for local MCP clients, but you can run the server with --transport http --port 3000 to expose an HTTP endpoint.
